node.js 설치 후 생활코딩으로 기본개념 복습 후 npm으로 모듈 설치

하루종일 구글링해서 찾아낸 sql연결법

require로 'mysql'을 불러와 변수 mysql에 저장.(아마 npm으로 받은 모듈에서 불러온게 아닐까 예상)

mysql모듈의 createConnection 함수와 connect함수를 작성.

  • host : 접속 주소를 나타낸다.
  • user : 유저 아이디.
  • password : 접속비밀번호.
  • database : 해당 호스트안에서의 DB를 선택.

connet 함수는 접속이 됐을때와 안됐을때의 동작을 정해주는 함수.

query함수로 select하여 원하는 값을 불러와 dataarr변수에 배열로 저장한다. 확인차 console.log로 출력해봄.

node로 mysql.js를 호출하여 확인했다. 원하는 값이 출력되는 것을 확인. 그렇다면 지금 dataarr변수에는 db에서 가져온 값이 물려있는 것.

하지만 문제는 지금부터 시작이었다.

기본기와 문법을 몰라 db값을 저장한 변수를 chart.js에 물리는 법을 수도없이 검색했지만 node.js로 설명해주는 곳은 없었고 (거의다 php만 쓰더라..) 결국 네카라쿠배 OT가 시작되는 7시 직전까지 검색만 죽어라 하다가 끝나버렸다.
